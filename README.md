# My Profile - Portfolio

A modern, responsive portfolio built with React, Vite, and Tailwind CSS.

## Features
- Hero section with animations
- About section with skill categories
- Featured projects with hover effects
- Contact form and social links
- Responsive design for mobile and desktop

## Deployment to Netlify
1. Connect your GitHub repository to Netlify.
2. The `netlify.toml` file in this project already configures the build settings:
   - **Build Command**: `npm run build`
   - **Publish Directory**: `dist`
3. Netlify will automatically detect these settings and deploy your site.

## Local Development
```bash
npm install
npm run dev
```

## Technologies Used
- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
